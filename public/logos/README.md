# 회사 로고 이미지

이 폴더에는 각 회사의 로고 이미지를 저장합니다.

## 파일명 규칙

- 파일명은 소문자와 하이픈(-)을 사용합니다.
- 예: `sk-ax.png`, `samsung-sds.png`, `lg-cns.png`

## 이미지 형식

- 형식: PNG (권장) 또는 SVG
- 크기: 최소 200x200px 이상 권장
- 배경: 투명 배경 권장

## 사용 방법

1. 로고 이미지 파일을 이 폴더에 추가합니다.
2. 파일명은 위의 규칙에 따라 지정합니다.
3. `CompanyLogo` 컴포넌트가 자동으로 로고를 표시합니다.
