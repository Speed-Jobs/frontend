# Speed Jobs - AI 기반 채용 인텔리전스 서비스

경쟁사 채용공고를 한눈에 파악하는 AI 기반 채용 인텔리전스 서비스입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 주요 기능

- **대시보드**: 실시간 경쟁사 공고와 트렌드 시각화
- **분석 리포트**: 채용 데이터 인사이트 제공
- **자동 매칭**: 경쟁사 공고와 직무 자동 매칭
- **공고 품질 평가**: AI 기반 공고 품질 개선 제안

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Recharts
